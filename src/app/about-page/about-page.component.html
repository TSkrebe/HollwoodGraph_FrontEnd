About

This project was created using an unusual technology stack. Front-end is written in Angular2/Typescript though final version was released only a few weeks ago(mid Sep 2016). In addition to Angular2 I also took advantage of Bootstrap3 and D3 visualization library. Probably the most unconventional choice was to pick Neo4j as database. It's graph database that is extremely easy to uses and treats relationships as first class citizens
You can find code on Github

Used technologies

Frontend:
  • Angular2
  • Bootstrap3
  • D3 library
Backend:
  • Flask
Database:
  • Neo4j
